# テキスト配置

note-mcpでは、段落のテキスト配置（中央寄せ、右寄せ、左寄せ）をMarkdown記法でサポートしています。

## 記法

| 記法 | 配置 | 例 |
|------|------|-----|
| `->テキスト<-` | 中央寄せ | `->中央に表示<-` |
| `->テキスト` | 右寄せ | `->右に表示` |
| `<-テキスト` | 左寄せ | `<-左に表示` |

## 使用例

### 中央寄せ

```markdown
->注目のお知らせ<-
```

### 右寄せ

```markdown
->2024年12月31日
```

### 左寄せ

```markdown
<-著者: 田中太郎
```

## インライン書式との組み合わせ

配置記法は他のMarkdown書式と組み合わせて使用できます。

```markdown
->**重要なお知らせ**<-

->[詳細はこちら](https://example.com)

->**強調された**テキスト<-
```

## 注意事項

### 行頭からの記述が必要

配置記法は行頭から記述する必要があります。行の途中に `->` や `<-` が含まれていても、配置として認識されません。

```markdown
# 正しい例
->中央寄せ<-

# 認識されない例
これは -> 中央寄せ <- ではありません
```

### コードブロック内での無効化

コードブロック（ ` ``` ` または `` ` `` ）内では配置記法は処理されず、そのまま表示されます。

```markdown
` ` `
->これは配置されません<-
` ` `
```

### 1行1配置

配置は段落（1行）単位で適用されます。複数行にまたがる配置はできません。

```markdown
# 正しい例
->1行目<-
->2行目<-

# 1行ずつ配置が適用される
```

## 技術的な詳細

### HTMLへの変換

配置記法はnote.comのエディタに送信する際、以下のHTMLに変換されます：

- 中央寄せ: `<p style="text-align: center">テキスト</p>`
- 右寄せ: `<p style="text-align: right">テキスト</p>`
- 左寄せ: `<p style="text-align: left">テキスト</p>`

### ラウンドトリップ変換

note.comから取得したHTMLは、自動的に配置記法を持つMarkdownに逆変換されます。これにより、記事の編集時に配置情報が保持されます。
