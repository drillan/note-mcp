# Ralph Wiggum Loop

Ralph Wiggum technique を使用してタスクを反復実行します。

## 使用方法

```
/ralph-loop "タスク内容"
```

## 実行

以下のスクリプトを実行してください：

```bash
/home/driller/.claude/plugins/cache/claude-plugins-official/ralph-wiggum/6d3752c000e2/scripts/setup-ralph-loop.sh "$ARGUMENTS" --completion-promise "COMPLETE" --max-iterations 255
```

タスク完了時は `<promise>COMPLETE</promise>` を出力してループを終了してください。
